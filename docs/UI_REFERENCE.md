# UI Reference

Mobile-first, age-friendly UI principles:

- Large readable text.
- Large buttons.
- One primary action per screen.
- Short copy.
- Low-pressure choices.
- Clear feedback after every action.
- No dense dashboards in the main flow.
- Warm but not childish visual tone.

Primary screens:

- Talk: large simulated voice input and otter presence.
- Breathe: simple breathing focus and short prompts.
- Move: gentle guided steps with stop-anytime framing.
- Timeline: readable record cards.
- Me: demo status, device simulator, privacy boundary.
